GENERAL TIPS

For solving this exercises be sure you have the command gulp serve running, it will compile the changes in your sass file and will refresh your browser automatically. Put your sass code inside the file mycss.scss to solve the exercises. Always put your html inside the exercise-wrapper div. Try to put the code of your solution inside a div wrapping them as we will reuse code.

EXERCISE REQUIREMENTS

Create 4 rectangles that ocupy the full with of the exercise wrapper(blue one) and 125px of height. Each of the rectangle should have a diferent colour, and 2px $black border, please use the classes i've provided for this purpose. The containers should be stacked and separated 15px from each other and other divs inside the main wrapper.

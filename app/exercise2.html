EXERCISE REQUIREMENTS

Reuse the container with the rectangles of exercise one. Now make the containers 50% with of exercise wrapper and separate and make them separate 30px from each other horizontally only in desktop resolutions (>1024px)

TIPS

Remember the float property will help you stack containers horizontally. Use the media query provided as example in mycss.scss. Use the property box-sizing: border-box that calculates a percentage including borders and paddings. Remember that margins are added to percentage althoug using box-sizing: border-box. Add the clearfix class(provided by me) to the rectangles container. It solves the problem of a container having 0 height if only contains floated elements.
